# mcutie
MQTT recon app

![alt demo](demo.gif)


## Setup instructions
* Install [mosquitto](https://mosquitto.org/download/)
* Project uses [pipenv](https://docs.pipenv.org/)
```sh
git clone https://github.com/aldnav/mcutie.git
cd mcutie
pipenv install
# other commands on the preview above
```
